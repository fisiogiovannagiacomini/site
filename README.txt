PUBLICAÇÃO RÁPIDA (SITE ONE-PAGE)

1) Estrutura
- index.html
- assets/
  - css/style.css
  - js/main.js
  - img/ (todas as imagens)

2) Como publicar
- Hospedagem comum:
  Envie a pasta inteira (index.html + assets) para o diretório público do servidor (geralmente "public_html" ou "www").
- GitHub Pages / Netlify / Vercel:
  Suba a pasta do site como projeto. O arquivo principal é o index.html.
